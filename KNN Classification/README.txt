Diabetes Detection:

Dataset Description:
In this problem given are Diabetes Data set consisting of following features ~
['Pregnancies', 'Glucose', 'BloodPressure', 'SkinThickness', 'Insulin', 'BMI', 'DiabetesPedigreeFunction', 'Age', 'Outcome']
and the task is to predict whether a person is suffering from diabetes or not (Binary Classification)

Task:
Classification Task, classify a person as 0 or 1 (Diabetic or Not) using K-Nearest Neighbors
classifier/Logistic regression.
